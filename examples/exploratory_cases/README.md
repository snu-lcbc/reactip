# Exploratory Cases

This directory contains research-only ReactIP inputs for exploratory SE-GSM and
single-structure MLIP studies.

Unlike the files under `benchmark_cases/`, these cases are not curated reaction
benchmarks. Their driving coordinates are hypotheses for probing model
capabilities and may or may not correspond to a chemically realistic pathway.

Current case:

- `2_4_diphenylpentane__C17H20`

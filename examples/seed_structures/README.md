# Supplemental Seed Structures

These structures are supplemental starting geometries only. They are useful for
calculator demonstrations and future SE-GSM experiments, but they are not
curated benchmark trajectories and are not used by the default SDF/GIF demo.

Provenance:

- tutorial page:
  https://mhyeok1.github.io/dand_docs/docs/how-to.html
- ZIP source:
  https://github.com/mhyeok1/dand_docs/raw/refs/heads/docs/assets/mother_structures_for_tut.Zip

The original archive contains five optimized mother structures:

- `Cl7138__C4H4ClNS`
- `Cl7164__C4H4ClNO`
- `Cl7166__C4H4ClNO`
- `Cl7168__C4H4ClNO`
- `Cl7188__C4H4ClNO`

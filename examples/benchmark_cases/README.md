This directory contains public-facing SE-GSM benchmark inputs for ReactIP demos.

Current benchmark case:

- `butadiene_ethylene_diels_alder__C6H10`

Source provenance:

- reactant fixture from
  `se-gsm/molecularGSM/TEST/dielsAlder/qchem/se-gsm/scratch/initial0001.xyz`
- driving coordinates from
  `se-gsm/molecularGSM/TEST/dielsAlder/qchem/se-gsm/ISOMERS0001`

does not require users to know the internal `se-gsm/` fixture layout.
